# 🌿 Nabatk — Modern Flower & Plant E-Commerce UI

A fully responsive, visually immersive front-end e-commerce website for an Egyptian flower and plant store.

Designed with a modern Green & White aesthetic, smooth animations, and a premium user experience — built entirely using **HTML, CSS, and Vanilla JavaScript**.

---

## 🎯 Project Overview

**Nabatk (نباتك)** is a complete front-end UI system for an online plant & flower store, including:

* Landing Page
* Shop / Products Page
* Product Details Page
* About Us Page
* Blog & Articles
* Contact Page
* Cart & Checkout Flow
* FAQ Page

This project focuses on **UI/UX excellence**, performance, and clean architecture without any external frameworks.

## 🧩 Features

### 💻 UI/UX

* Fully responsive (Mobile-first)
* Modern glassmorphism navbar
* Smooth animations (CSS + Intersection Observer)
* Clean grid layouts
* Interactive components

### 🛍️ E-Commerce Simulation

* Product filtering & sorting
* Cart system باستخدام `localStorage`
* Wishlist functionality ❤️
* Dynamic product pages

### ⚡ Interactivity

* Image gallery with preview switching
* Animated counters & carousels
* Newsletter subscription UI
* Form validation (Contact & Checkout)

### 🧠 Advanced Frontend Concepts

* Vanilla JS state management
* DOM manipulation
* Intersection Observer API
* CSS animations & transitions
* Responsive design patterns
